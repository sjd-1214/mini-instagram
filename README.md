# Mini Instagram

Mini Instagram is a project implemented in C++ that mimics the core functionalities of a social media platform, allowing users to interact through posts, likes, and comments. This project serves as a simplified version of Instagram, focusing on essential features.



## Features

- **User Management**: 
  - Create and manage user profiles.
  - Log in and out of user accounts.

- **Post Management**:
  - Create, edit, and delete posts.
  - View posts of other users.

- **Social Interactions**:
  - Like and comment on posts.



## Installation

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/ajmalrazaqbhatti/mini-instagram.git
   ```
2. Navigate to the project directory:
   ```bash
   cd mini-instagram
   ```
3. Compile the project using a C++ compiler:
   ```bash
   g++ -o mini-instagram *.cpp
   ```
4. Run the executable:
   ```bash
   ./mini-instagram
   ```



## Usage

- Follow the instructions provided in the console interface to explore the features of Mini Instagram.
- Users can create accounts, log in, create posts, and interact with other users' posts.



## Project Structure

- The project is written entirely in C++.
- Core components include:
  - User management
  - Post creation and management
  - Social interaction handling
